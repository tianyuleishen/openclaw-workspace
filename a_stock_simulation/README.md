# A股AI策略模拟交易系统

## 项目概述

A股AI策略模拟交易系统是一个基于人工智能算法的A股市场模拟交易平台。系统集成了多因子AI策略、风险管理机制和实时数据分析功能，旨在通过模拟交易验证AI策略的有效性。

## 系统架构

### 核心组件

1. **基础模拟器** (`stock_simulator.js`)
   - 资金管理与交易执行
   - 投资组合跟踪
   - 交易历史记录

2. **高级策略引擎** (`advanced_strategy.js`)
   - 多因子模型（动量、价值、质量、波动率、流动性、情绪）
   - 基本面分析
   - 市场状态感知

3. **优化交易系统** (`fixed_optimized_simulator.js`)
   - 动态止盈止损
   - 资金管理
   - 交易成本模型

4. **增强版模拟器** (`enhanced_simulator_with_tushare.js`)
   - 集成真实市场数据
   - 改进的风险控制
   - 实时交易信号

5. **调度器** (`daily_trading_scheduler.js`)
   - 自动化每日交易
   - 状态管理
   - 性能监控

6. **控制台** (`trading_console.js`)
   - 交互式管理界面
   - 实时监控
   - 报告生成

## 系统配置

### 策略参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 初始资金 | ¥100,000 | 起始模拟资金 |
| 最大持仓 | 8只 | 同时持有的股票数量 |
| 单次风险 | 3% | 单次交易风险占总资金比例 |
| 止损线 | 8% | 单股亏损达到此比例时止损 |
| 止盈线 | 15% | 单股盈利达到此比例时止盈 |
| 最小成交量 | 1,000,000 | 股票日成交量要求 |

### 交易品种

## 性能监控

### 评估指标

- **总收益率**: 总体投资回报率
- **年化收益率**: 年化投资回报率
- **夏普比率**: 风险调整后收益
- **最大回撤**: 最大资金回撤幅度
- **胜率**: 盈利交易占比
- **Alpha**: 超额收益

### 日志管理

系统自动生成以下日志：
- 每日交易日志 (`trade_log_YYYY-MM-DD.json`)
- 系统状态快照 (`current_state.json`)
- 交易历史记录
- 性能分析报告

## 使用说明

### 1. 单次交易测试
使用控制台选项 `3` 执行单次模拟交易，验证策略效果。

### 2. 自动交易调度
使用控制台选项 `1` 启动自动交易调度器，系统将在每个交易日自动执行交易。

### 3. 性能监控
使用控制台选项 `5` 查看详细性能报告，监控策略表现。

### 4. 状态管理
系统会自动保存和恢复状态，确保交易连续性。
