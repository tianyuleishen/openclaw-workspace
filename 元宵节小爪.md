## 🎥 图生视频（首帧模式）

**提示词**（参考阿里万相最佳实践）:
```
[镜头动作] 小爪从画面中心开心地跳起来，身体轻盈地旋转，双脚交替跳跃
[表情细节] 大眼睛笑得眯成月牙，嘴巴张大开心地笑，露出兴奋的表情，头上的触须随着跳跃摇摆
[肢体动作] 举着灯笼的爪子摇摆摇晃，另一只爪子不时挥手打招呼，尾巴开心地左右摇摆
[环境氛围] 元宵节灯笼在周围轻轻飘动，金色光点环绕飞舞，背景有温暖的节日光晕，整体氛围活泼欢快
[节奏韵律] 动作从慢到快，跳跃幅度逐渐增大，保持15秒的持续欢快节奏
[声音同步] 配合欢快的背景音乐，动作节奏与音乐节拍吻合
```

**关键要素**:
- ✅ 镜头动作：从跳起到旋转
- ✅ 表情细节：笑眼、嘴巴、触须
- ✅ 肢体动作：爪子、尾巴摇摆
- ✅ 环境氛围：灯笼、光点环绕
- ✅ 节奏韵律：15秒持续欢快

## 💡 提示词编写要点（参考阿里万相）

**结构公式**:
```
提示词 = 镜头动作 + 表情细节 + 肢体动作 + 环境氛围 + 节奏韵律 + 声音同步
```

**关键原则**:
1. **具体动作**: 避免"开心"，要用"跳起来"、"旋转"
2. **表情描述**: 具体到眼睛、嘴巴、触须
3. **环境细节**: 灯笼、光点、背景
4. **节奏控制**: 15秒动作分解
