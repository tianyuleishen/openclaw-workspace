# 推理引擎配置

## 自动启用

从 v5.9 开始，推理引擎会自动集成到所有会话中。

## 配置

### 问题类型自动检测

| 类型 | 关键词 | 推理引擎 |
|------|--------|---------|
| 三角函数 | tan, cos, sin, θ | v5.9 |
| 极值组合 | 最大, 最小, 范围, 格子 | v5.7 |
| 函数直线 | 函数, 斜率, 共线 | v5.8 |
| 几何 | 三角形, 圆形, 角度 | v5.5 |

### 使用方法

```python
from reasoning_engine_integrator import solve

# 自动分析并返回答案
answer = solve("你的问题")
```
