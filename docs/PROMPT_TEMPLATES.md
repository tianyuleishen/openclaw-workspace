## 💡 提示词编写技巧

### 核心要素

| 要素 | 说明 | 示例 |
|------|------|------|
| **场景** | 描述主要画面 | 办公室、会议室、街道 |
| **人物** | 人物状态 | 年轻白领、程序员、学生 |
| **动作** | 发生什么 | 工作、学习、讨论 |
| **风格** | 画面风格 | 科技感、温馨、励志 |
| **色调** | 颜色主题 | 蓝色橙色、暖色调 |
| **节奏** | 剪辑节奏 | 快节奏、舒缓 |
| **配音** | 声音风格 | 男声、女声、专业 |
| **时长** | 视频时长 | 20-30秒 |
| **结尾** | 结束方式 | 字幕、号召性文字 |

### 编写口诀

```
场景人物要清晰，
动作节奏要合理。
风格色调要统一，
配音时长要考虑。
结尾号召不能忘，
痛点共鸣最给力！
```

## 🎨 高质量提示词公式

```
[场景描述] + [人物动作] + [风格色调] + [节奏要求] + [结尾字幕] + [配音风格] + [时长] + [背景音乐]
```

```
现代办公室场景（场景），

最后字幕显示"AI让效率提升10倍"（结尾），

25秒（时长），
轻快电子背景音乐（背景音乐）。
```

## 📋 使用说明

1. **选择模板**：根据内容类型选择对应文件夹
2. **修改参数**：调整场景、人物、时长等参数
3. **测试效果**：先生成几个视频测试效果
4. **优化调整**：根据实际效果调整提示词
5. **批量生成**：准备好提示词列表，一键生成

---

## 🚀 批量生成命令

```bash
# 生成单个视频
python3 wanxiang_video.py --prompt "你的提示词"

# 批量生成（创建 prompts.txt 文件，每行一个提示词）
python3 wanxiang_video.py --batch prompts.txt
```
