## 五、具体安装步骤

### 5.1 aisa-media-gen 安装

**步骤1：检查环境**
```bash
# 检查 Node.js
node --version  # 需 v18+

# 检查 npm
npm --version
```

**步骤2：安装技能**
```bash
# 安装
clawhub install aisa-media-gen

# 配置
cd ~/.openclaw/skills/aisa-media-gen
cat README.md
```

**步骤3：配置 API Key**
```bash
# 设置环境变量
export AISA_API_KEY="your-api-key-here"

# 或在配置文件中设置
cat config.yaml
```

**步骤4：测试**
```bash
# 运行测试
npm test

# 生成第一个视频
npm run generate -- --prompt "一个科技感短视频"
```

---

### 5.2 veo 安装

**步骤1：注册 Google Cloud**
```
1. 访问 https://console.cloud.google.com
2. 点击 "开始免费试用"
3. 填写信息
4. 绑定支付方式（需要信用卡）
```

**步骤2：启用 Veo API**
```
1. 在搜索框搜索 "Veo API"
2. 点击启用
3. 了解价格（按量付费）
```

**步骤3：创建 API Key**
```
1. 进入 "API 和服务" → "凭证"
2. 点击 "创建凭证" → "API Key"
3. 复制 Key
4. 限制 Key（可选）
```

**步骤4：安装技能**
```bash
# 安装
clawhub install veo

# 配置
export GEMINI_API_KEY="your-google-api-key"
export VEO_MODEL="veo-1"
```

## 六、常见问题

### Q1：需要服务器吗？

**答：视情况而定**

| 场景 | 需要服务器 | 说明 |

| 公网访问 | ✅ 需要 | 接收 Webhook |
| 规模化运营 | ✅ 需要 | 性能需求 |

**推荐配置**：
```
最低配置：
├── CPU: 1 核心
```
