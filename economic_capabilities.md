## 📊 当前经济知识库

### 1. 有效市场假说 (EMH)

**内容**:
```
有效市场假说 (Efficient Market Hypothesis)

三种形式:
1. 弱式有效市场 (Weak Form)
   - 股价反映所有历史价格信息
   - 技术分析无效
   - 投资者无法通过历史价格获利

2. 半强式有效市场 (Semi-Strong Form)
   - 股价反映所有公开信息
   - 基本面分析无效
   - 公开信息立即反映在价格中

3. 强式有效市场 (Strong Form)
   - 股价反映所有信息（包括内幕）
   - 即使内幕交易也无法获利
   - 现实中不存在
```

**来源**: 《金融学基础》、Fama (1970)

### 2. 行为金融学挑战

**内容**:
```
行为金融学对EMH的挑战:

1. 过度自信
   - 投资者高估自己的预测能力
   - 交易频繁，手续费侵蚀收益

2. 羊群效应
   - 投资者模仿他人行为
   - 导致价格偏离基本面

3. 损失厌恶
   - 亏损带来的痛苦大于盈利的快乐
   - 投资者过早卖出盈利股票
   - 过晚卖出亏损股票

4. 锚定效应
   - 过度依赖初始信息
   - 无法及时调整预期

5. 市场异象
   - 小市值效应
   - 价值效应
   - 动量效应
   - 这些现象无法用EMH解释
```

**来源**: 《行为金融学》、Kahneman & Tversky (1979)

### 3. IS-LM模型

**内容**:
```
IS-LM模型:

IS曲线 (Investment-Savings):
- 表示商品市场均衡
- Y = C + I + G + NX
- 利率与收入负相关

LM曲线 (Liquidity-Money):
- 表示货币市场均衡
- M/P = kY - hi
- 收入与利率正相关

均衡点:
- IS与LM交点决定均衡利率和收入
- 政府可以通过财政政策移动IS曲线
- 央行可以通过货币政策移动LM曲线
```

**来源**: 《宏观经济学》、Hicks (1937)

### 4. AS-AD模型

**内容**:
```
AS-AD模型 (总供给-总需求):

AD曲线 (总需求):
- P与Y负相关
- 由消费、投资、政府支出决定

AS曲线 (总供给):
- 短期：向上倾斜（粘性工资）
- 长期：垂直（古典学派）

政策影响:
- 扩张性AD政策：短期增加Y，提高P
- 长期只有价格水平上升
```

```

- 短期+长期分析

   - AS-AD：价格可以变化

   - IS-LM：财政政策效果明显
   - AS-AD：长期只有价格效应
```
