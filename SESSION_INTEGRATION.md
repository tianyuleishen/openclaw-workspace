# 会话集成配置

## 推理引擎自动启用

从 2026-02-11 开始，所有会话自动启用推理引擎 v5.9。

## 工作原理

### 自动问题检测

```
用户消息
    │
    ├─ 关键词检测
    ├─ 问题类型分类
    └─ 调用对应推理引擎
```

### 支持的问题类型

| 类型 | 关键词 | 示例 |
|------|--------|------|
| 三角函数 | tan, cos, sin, θ | tanθ₁·...·tanθₙ = 2^(n/2) |
| 极值组合 | 最大, 最小, 范围 | 100×100格子，每种颜色≤10000 |
| 函数直线 | 函数, 斜率, 共线 | y=(x+1)/(\|x\|+1) |
| 几何 | 三角形, 圆形, 角度 | 翻折, 内切圆 |

## 使用方法

### 快捷调用

```python
from reasoning_engine_integrator import solve

# 自动分析
answer = solve("你的数学问题")
```

### 直接导入

```python
from reasoning_engine_v5_9 import ReasoningEngineV5_9

engine = ReasoningEngineV5_9()
result = engine.analyze_function_problem({...})
```
