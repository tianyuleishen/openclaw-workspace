## 🚀 常用命令

### 1. 传媒板块实时行情

```bash
python3 stock_query.py
# 或
python3 stock_tools.py quote
```

**输出**:
```
📺 传媒板块实时行情
⏰ 时间: 2026-02-09 22:00
============================================================
名称           价格         涨跌幅       
------------------------------------------------------------
中文在线         35.28        +20.00% ✅ 涨停
荣信文化         42.24        +20.00% ✅ 涨停
============================================================
```

### 2. 查询指定股票

```bash
# 贵州茅台
python3 stock_query.py sh600519

# 中文在线
python3 stock_query.py sz300364

# 上证指数
python3 stock_query.py sh000001

# 创业板指
python3 stock_query.py sz399006
```

### 3. 获取涨停数据

```bash
python3 get_limit_up.py
# 或
python3 stock_tools.py limitup
```

**输出**: 
- 生成 `limit_up_stocks_YYYYMMDD.json` 文件
- 包含所有涨停股票

### 4. AI多因子分析

```bash
python3 stock_analyzer.py
# 或
python3 stock_tools.py analyze
```

**分析维度**:
- 动量因子
- 价值因子
- 质量因子
- 波动率因子
- 流动性因子
- 情绪因子

### 5. 涨停原因分析

```bash
python3 stock_rally_analyzer.py
# 或
python3 stock_tools.py reason
```

## 🔍 故障排除

### 问题1: 获取数据失败

**可能原因**: 网络连接问题

**解决**:
```bash
# 测试腾讯财经API
curl https://qt.gtimg.cn/q=sh000001
```

### 问题2: 解析错误

**解决**: 确保使用最新版本的脚本

```bash
# 重新下载脚本
python3 stock_query.py
```

### 问题3: 股票代码不存在

**解决**: 检查股票代码格式

```bash
# 正确格式
```
