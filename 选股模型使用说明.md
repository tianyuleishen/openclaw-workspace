## 评分标准（40分及格，75分强烈推荐）

| 条件 | 分数 | 说明 |
|------|------|------|
| 市场均涨幅>1% | +15 | 大盘环境好 |
| 市场均涨幅>0% | +8 | 大盘环境一般 |
| 涨停(≥9.5%) | +30 | 强烈看涨 |
| 涨幅≥7% | +22 | 强势股 |
| 涨幅≥5% | +18 | 较强股 |
| 涨幅≥3% | +12 | 稳健股 |
| 涨停股换手特殊加分 | +8 | 涨停特殊处理 |
| 换手率3-15% | +10 | 筹码活跃 |
| 收盘>开盘 | +5 | 分时强势 |

## 注意事项

1. **Token配置**: 使用Tushare Pro Token
2. **API限制**: 免费版每分钟500次请求
3. **数据延迟**: 可能获取的是历史交易日数据
4. **市场环境**: 弱势市场可能结果较少
