# 小爪智能量化交易系统 v1.0

## 📋 系统概述

基于TuShare Pro真实数据的AI量化交易系统，实现：
- ✅ 选股：多因子评分模型
- ✅ 买入：技术指标 + 仓位管理
- ✅ 卖出：止盈止损 + 信号判断
- ✅ 风控：风险收益比控制

---

## 🎯 核心策略

### 选股因子（权重分配）

| 因子 | 权重 | 说明 |
|------|------|------|
| 动量因子 | 30% | 5日/10日/20日涨幅趋势 |
| 强度因子 | 25% | 价格相对近期高点的位置 |
| 量能因子 | 20% | 量比（放量健康） |
| 位置因子 | 15% | 价格在震荡区间的位置 |
| 涨跌幅 | 10% | 当日涨跌幅 |

### 评分规则

```
综合评分 = 基础分50 + 动量调整 + 强度调整 + 量能调整 + 位置调整 + 涨跌幅调整
```

---

## 📊 买入信号判断

### 买入条件（满足≥5条）

1. ✅ 综合评分 ≥ 70分
2. ✅ 5日动量 > +3%
3. ✅ 强度因子 > 0.6
4. ✅ 量比在 1.0~3.0 之间
5. ✅ 价格位置 ≥ 40%
6. ✅ 今日涨幅 > 0%

### 关注条件（满足≥2条）

1. ✅ 综合评分 ≥ 55分
2. ✅ 5日动量 > 0
3. ✅ 今日涨幅 > 0%

### 卖出条件（满足≥3条）

1. ❌ 综合评分 < 30分
2. ❌ 5日动量 < -5%
3. ❌ 今日跌幅 > -5%
4. ❌ 价格位置 < 20%

---

## 💰 仓位管理

### 风控参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 最大仓位 | 50% | 单只股票最多占总资金50% |
| 最小买入 | ¥5000 | 低于此金额不买入 |
| 单笔风险 | 2% | 单笔最大亏损限制 |
| 止损比例 | -8% | 跌破止损价卖出 |
| 止盈比例 | +15% | 达到止盈价卖出 |
| 风险收益比 | 1.9:1 | 预期收益/风险 |

### 仓位计算公式

```
买入金额 = 总资金 × 50% × (评分/100 + 0.5)
```

示例：¥10万资金，评分80分
```
买入金额 = 100,000 × 50% × (0.8 + 0.5) = ¥65,000
```

---

## 📈 支撑压力位计算

- **支撑位**: 20日低点 + 23.6%回撤位
- **压力位**: 20日高点
- **止损价**: 当前价格 × (1 - 8%)
- **止盈价**: 当前价格 × (1 + 15%)
