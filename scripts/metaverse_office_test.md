## 🔧 生成步骤

### 步骤1: 生成首帧图片
1. 访问: https://tongyi.aliyun.com/wanxiang/
2. 选择: 文生图
3. 输入首帧提示词
4. 设置: 尺寸 720x1280，风格 动漫
5. 生成并下载

### 步骤2: 生成末帧图片
1. 选择: 文生图
2. 输入末帧提示词
3. 设置: 尺寸 720x1280，风格 动漫
4. 生成并下载

### 步骤3: 生成视频
1. 选择: 图生视频 -> 首尾帧模式
2. 上传首帧图片
3. 上传末帧图片
4. 设置: 时长 15秒，分辨率 720p
5. 生成并下载
