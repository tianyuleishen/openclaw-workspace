| 类型 | 示例 | 答案 | 置信度 |

## 使用方法

```python
from skills.reasoning import solve

# 因式分解
solve("a²(b - c) + b²(a - c) + c²(a - b) 因式分解")
# 答案: (a-b) × (b-c) × (c-a)

# 三角函数
solve("tanθ₁·...·tanθₙ = 2^(n/2)")
# 答案: λ = n-1

# 极值
solve("100×100格子，每种颜色≤10000")
# 答案: 12
```
