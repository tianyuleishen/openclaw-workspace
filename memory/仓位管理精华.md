## 1️⃣ 什么是仓位管理

### 定义

> "仓位管理是指投资者在交易过程中对资金的分配和运用,旨在控制风险、保护本金和提高收益率."

**核心**: 风险量化，合理分配资金

---

### 仓位管理的构成

| 构成 | 说明 |
|------|------|
| **资金分配** | 如何分配资金 |
| **持仓比例** | 每只股票的仓位 |
| **加减仓** | 如何加减仓 |
| **风险控制** | 如何控制风险 |

---

## 2️⃣ 仓位管理的重要性

### 为什么需要仓位管理

| 重要性 | 说明 |
|--------|------|
| **控制风险** | 避免单次交易过大损失 |
| **保护本金** | 本金是交易的基础 |
| **提高收益** | 合理配置提高收益 |
| **心态稳定** | 仓位适中，心态更好 |

---

### 仓位与风险

| 仓位比例 | 风险等级 | 心理压力 |
|---------|---------|---------|
| **满仓** | 🔴 极高 | 极大 |
| **80%** | 🔴 高 | 很大 |
| **50%** | 🟡 中等 | 中等 |
| **30%** | 🟢 较低 | 较小 |
| **10%** | 🟢 低 | 很小 |

## 3️⃣ 仓位管理原则

### 核心原则

| 原则 | 说明 |
|------|------|
| **分散投资** | 不把鸡蛋放在一个篮子里 |
| **风险可控** | 单次风险可控 |
| **动态调整** | 根据市场调整仓位 |
| **纪律执行** | 严格执行仓位计划 |

---

### 仓位管理铁律

| 铁律 | 要求 |
|------|------|
| **单股不超50%** | 分散风险 |
| **单次不超20%** | 控制单次风险 |
| **总仓位可控** | 根据市场调整 |
| **保留现金** | 留有抄底资金 |

## 4️⃣ 仓位配置方法

### 方法1: 固定比例法

**配置公式**:

| 类型 | 比例 | 说明 |
|------|------|------|
| **核心仓位** | 30-40% | 确定性高的标的 |
| **卫星仓位** | 30-40% | 成长性标的 |
| **现金** | 20-30% | 等待机会 |

---

### 方法2: 市场环境法

**根据市场调整**:

| 市场环境 | 仓位 | 说明 |
|---------|------|------|
| **牛市** | 80-100% | 积极做多 |
| **震荡** | 50-70% | 波段操作 |
| **熊市** | 20-40% | 谨慎防守 |
| **恐慌** | 0-20% | 观望为主 |

---

### 方法3: 确定性法

**根据确定性配置**:

| 确定性 | 仓位 | 说明 |
|-------|------|------|
| **高确定性** | 40-50% | 龙头股、主升浪 |
| **中确定性** | 20-30% | 补涨股、趋势股 |
| **低确定性** | 10-20% | 试仓、观望 |

---

### 方法4: 分批建仓法

**建仓节奏**:

| 批次 | 仓位 | 买入条件 |
|------|------|---------|
| **第一仓** | 30% | 支撑位确认 |
| **第二仓** | 30% | 反弹确认 |
| **第三仓** | 40% | 趋势反转 |

## 5️⃣ 风险控制

### 止损策略

> "合理的仓位管理和风险控制是成功的关键"

**止损原则**:

| 类型 | 设置 | 说明 |
|------|------|------|
| **固定止损** | -5%到-8% | 单股最大亏损 |
| **时间止损** | 3-5天 | 不涨就卖 |
| **均线止损** | 跌破5日/10日线 | 趋势止损 |

---

### 仓位与止损

| 仓位 | 止损幅度 | 建议 |
|------|---------|------|
| **10%** | -10% | 可接受 |
| **20%** | -5% | 严格止损 |
| **30%** | -3% | 及时止损 |
| **50%** | -2% | 果断止损 |

---

### 风险控制清单

| 检查项 | 要求 |
|--------|------|
| **单股仓位** | 不超过50% |
| **单次风险** | 不超过总资金2% |
| **总风险** | 日亏损不超过5% |
| **周亏损** | 周亏损不超过15% |
