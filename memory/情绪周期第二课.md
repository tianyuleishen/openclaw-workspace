## 1️⃣ 情绪周期的本质

### 什么是情绪周期

> "情绪周期是市场情绪从低迷到高涨再到低迷的循环过程"

**情绪周期**是指市场整体情绪在乐观和悲观之间循环往复的过程。

---

### 情绪周期的本质

| 本质 | 说明 |
|------|------|
| **人性循环** | 贪婪与恐惧的循环 |
| **资金循环** | 资金进出市场的循环 |
| **预期循环** | 预期好坏的循环 |
| **行为循环** | 追涨杀跌的循环 |

---

### 情绪周期的特点

| 特点 | 说明 |
|------|------|
| **周期性** | 反复循环 |
| **对称性** | 上涨与下跌对称 |
| **自我强化** | 情绪会自我强化 |
| **非线性** | 不是线性发展 |

---

## 2️⃣ 情绪周期的阶段

### 经典七阶段

| 阶段 | 特征 | 操作策略 |
|------|------|----------|
| **冰点期** | 市场极度低迷 | 观察等待 |
| **复苏期** | 情绪开始回暖 | 小仓位试探 |
| **回暖期** | 情绪明显好转 | 逐步加仓 |
| **高潮期** | 市场极度亢奋 | 持股待涨 |
| **过热期** | 风险快速累积 | 逐步减仓 |
| **退潮期** | 情绪开始衰退 | 快速离场 |
| **冰点期** | 市场极度低迷 | 空仓等待 |

---

### 短线情绪周期

| 阶段 | 特征 | 操作 |
|------|------|------|
| **启动期** | 新题材出现 | 积极参与 |
| **发酵期** | 板块扩散 | 找龙头买 |
| **高潮期** | 板块大涨 | 持股待涨 |
| **分化期** | 板块分化 | 去弱留强 |
| **退潮期** | 板块下跌 | 空仓休息 |

---

### 日内情绪周期

| 时间段 | 情绪特征 | 操作 |
|--------|----------|------|
| 竞价期 | 情绪酝酿 | 观察为主 |
| 开盘期 | 情绪释放 | 快速反应 |
| 盘中期 | 情绪演化 | 跟随操作 |
| 收盘期 | 情绪总结 | 复盘计划 |

---

## 3️⃣ 情绪周期的判断

### 判断指标

| 指标 | 说明 | 数值判断 |
|------|------|----------|
| **涨停数量** | 涨停股票数量 | 多=高潮，少=冰点 |
| **跌停数量** | 跌停股票数量 | 多=退潮，少=回暖 |
| **连板高度** | 最高连板数 | 高=高潮，低=冰点 |
| **炸板率** | 涨停被打开比例 | 高=分化，低=一致 |
| **昨日涨停表现** | 昨日涨停今日表现 | 涨=强势，跌=弱势 |
| **成交量** | 市场成交量 | 放量=活跃，缩量=低迷 |

---

### 判断方法

#### 方法一：涨停数量判断

| 涨停数量 | 情绪阶段 | 操作策略 |
|----------|----------|----------|
| >100只 | 高潮期 | 持股待涨 |
| 50-100只 | 发酵期 | 积极参与 |
| 20-50只 | 回暖期 | 逐步参与 |
| <20只 | 冰点期 | 观察等待 |

---

#### 方法二：连板高度判断

| 连板高度 | 情绪阶段 | 操作策略 |
|----------|----------|----------|
| >10板 | 过热期 | 减仓止盈 |
| 7-10板 | 高潮期 | 持股待涨 |
| 4-6板 | 发酵期 | 积极参与 |
| 1-3板 | 启动期 | 小仓位试探 |
| 0板 | 冰点期 | 观望 |

---

#### 方法三：炸板率判断

| 炸板率 | 情绪阶段 | 操作策略 |
|----------|----------|----------|
| >50% | 退潮期 | 减少操作 |
| 30-50% | 分化期 | 去弱留强 |
| <30% | 高潮期 | 积极参与 |
| <20% | 发酵期 | 全力参与 |

## 4️⃣ 情绪周期与操作

### 不同阶段的操作

| 阶段 | 操作策略 | 仓位 |
|------|----------|------|
| **冰点期** | 观察等待 | 空仓 |
| **复苏期** | 小仓位试探 | 10-20% |
| **回暖期** | 逐步加仓 | 30-50% |
| **高潮期** | 持股待涨 | 60-80% |
| **过热期** | 逐步减仓 | 40-60% |
| **退潮期** | 快速离场 | 0-20% |

---

### 操作原则

| 原则 | 说明 |
|------|------|
| **顺势而为** | 顺应情绪周期操作 |
| **逆向思维** | 冰点买、高潮卖 |
| **控制节奏** | 不追涨杀跌 |
| **严格止损** | 破位必须止损 |
| **复盘总结** | 每天复盘总结 |

## 5️⃣ 情绪周期的切换

### 切换信号

| 信号 | 说明 |
|------|------|
| **新题材出现** | 可能开启新周期 |
| **老题材结束** | 周期即将切换 |
| **情绪反转** | 情绪快速转变 |
| **资金切换** | 资金从一个板块切换到另一个 |

---

### 切换时机

| 时机 | 说明 |
|------|------|
| **冰点转复苏** | 新题材启动 |
| **高潮转过热** | 风险累积 |
| **过热转退潮** | 资金出逃 |
| **退潮转冰点** | 情绪低迷 |

---

### 切换应对

| 切换 | 应对策略 |
|------|----------|
| **向上切换** | 加仓参与 |
| **向下切换** | 减仓避险 |
| **板块切换** | 跟随资金 |
| **风格切换** | 调整策略 |
