- 🌟 A股顶级游资

**核心思想**:
> "本人的体系是建立在对市场情绪的把握之上，寻找热点板块的龙头股进行操作"

**养家心法四大支柱**:
| 支柱 | 说明 |
|------|------|
| **情绪为王** | 一切围绕情绪 |
| **只做龙头** | 只做板块最强股 |
| **仓位管理** | 根据情绪控制仓位 |
| **心态稳定** | 保持良好操作心态 |

**养家经典语录**:
> "股票走势本质是持币者与持筹者的博弈产生，应该多揣摩两者之心，让自己站在主动的一方"
