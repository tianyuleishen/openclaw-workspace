## 1️⃣ 什么是龙头股

### 定义

> "龙头战法是老股民在股市征战十年的精髓,核心思想是:快、狠、准、捕食."

**龙头股**是指在某一板块或题材中涨幅最大、影响力最强、带动性最好的股票。

---

### 龙头股的作用

| 作用 | 说明 |
|------|------|
| **领涨** | 率先启动，带动板块 |
| **领跌** | 率先调整，影响板块 |
| **风向标** | 板块炒作的风向标 |
| **溢价** | 享受更高的市场溢价 |

---

### 龙头股的分类

| 类型 | 特点 | 操作难度 |
|------|------|---------|
| **市场总龙头** | 全市场最强 | 高 |
| **板块龙头** | 板块内最强 | 中 |
| **题材龙头** | 题材内最强 | 中 |
| **补涨龙头** | 补涨行情中 | 低 |

---

## 2️⃣ 龙头股特征

### 核心特征

> "龙头个股必须从涨停板开始.不能涨停的股票不可能做龙头."

**核心特征**:

| 特征 | 说明 |
|------|------|
| **涨停启动** | 连续涨停或连板 |
| **涨幅最大** | 板块内涨幅第一 |
| **成交量大** | 换手率高 |
| **带动性强** | 联动板块上涨 |
| **持续性强** | 炒作周期长 |

---

### 选股标准

**基本面标准**:

| 标准 | 要求 |
|------|------|
| **业绩** | 业绩不差，避免亏损 |
| **题材** | 题材正宗，受资金认可 |
| **行业** | 行业地位领先 |
| **市值** | 市值适中（50-500亿） |

**技术面标准**:

| 标准 | 要求 |
|------|------|
| **形态** | 形态突破或强势整理 |
| **均线** | 均线多头排列 |
| **成交量** | 成交量持续放大 |
| **位置** | 相对低位启动 |

## 3️⃣ 龙头股选择技巧

### 选股方法

#### 方法1: 涨停板筛选

> "一般只选5%以上换手的股票"

**筛选条件**:
1. ✅ 今日涨停（最先涨停的）
2. ✅ 换手率 > 5%
3. ✅ 成交量放大
4. ✅ 题材正宗

---

#### 方法2: 板块联动法

**操作步骤**:
1. 📊 发现板块异动
2. 🔍 找到领涨个股
3. ✅ 确认为板块龙头
4. 🎯 择机介入

---

#### 方法3: 题材发酵法

**判断依据**:
| 信号 | 说明 |
|------|------|
| **消息刺激** | 重大利好消息 |
| **政策出台** | 政策利好 |
| **行业景气** | 行业周期向上 |
| **资金流入** | 主力资金持续流入 |

---

### 龙头股识别技巧

| 技巧 | 说明 |
|------|------|
| **最先涨停** | 板块内第一个涨停 |
| **换手最大** | 换手率最高 |
| **成交最大** | 成交量最大 |
| **封单最大** | 涨停封单最大 |
| **带动最强** | 板块联动最紧密 |

**条件**:

- 💰 量比 > 2

- 🎯 题材正宗

#### 2. 开盘买入

**条件**:
- ⏰ 9:30-10:00
- 🚀 快速涨停
- 📊 板块联动
- 💰 换手充分

---

#### 3. 回调买入

**条件**:
- 📉 回调到支撑位
- ✅ 止跌企稳
- 📊 量能萎缩
- 🎯 均线支撑
