## 1️⃣ 盯盘基础

### 盯盘重要性

> "超短打板交易对盯盘要求比较高。"

**核心目标**:
- 🎯 识别龙头
- 🔥 发现热点
- ⚡ 把握时机

---

### 盯盘要点

#### 1. 观察对象

| 观察对象 | 关注点 | 目的 |
|---------|-------|------|
| **涨跌幅榜** | 排名变化 | 发现热点 |
| **成交额榜** | 大单流向 | 判断资金 |
| **换手率榜** | 活跃程度 | 识别异动 |
| **龙虎榜** | 主力动向 | 跟随资金 |

#### 2. 盯盘时间

| 时间段 | 重点 | 说明 |
|-------|------|------|
| **9:15-9:25** | 集合竞价 | 抢筹/出货 |
| **9:30-10:00** | 强势股 | 确定日内方向 |
| **10:00-11:30** | 盘中异动 | 发现机会 |
| **13:00-14:30** | 午后表现 | 确认强度 |
| **14:30-15:00** | 尾盘动作 | 抢筹/出货 |

## 2️⃣ 看盘界面设置

### 核心设置

> "手把手教你设置，打板快人一步！"

#### 1. 基础面板

**必备窗口**:

1. **涨跌幅排名** - 发现热点
2. **涨速排名** - 捕捉异动
3. **成交额排名** - 跟踪大资金
4. **换手率排名** - 识别活跃股

#### 2. 分屏设置

**推荐布局**:

```
┌─────────────────┬─────────────────┐
│   涨跌幅榜      │   涨速榜        │
├─────────────────┼─────────────────┤
│   成交额榜      │   换手率榜     │
├─────────────────┼─────────────────┤
│   自选股       │   大盘指数     │
└─────────────────┴─────────────────┘
```

#### 3. 盯盘精灵

**短线精灵设置**:

| 指标 | 设置 | 作用 |
|------|------|------|
| **涨速** | >3% | 捕捉异动 |
| **涨幅** | >5% | 发现强势 |
| **量比** | >2 | 识别放量 |
| **换手** | >10% | 跟踪活跃 |

## 3️⃣ 打板技巧

### 打板类型

#### 1. 首板

**特点**:
- 🔥 首次涨停
- ⚡ 风险较高
- 💰 收益也高

**打板条件**:
1. ✅ 板块联动
2. ✅ 题材新颖
3. ✅ 资金抢筹
4. ✅ 形态突破

---

#### 2. 连板

**特点**:
- 🚀 连续涨停
- 🛡️ 确定性高
- 💰 收益更高

**打板条件**:
1. ✅ 板块龙头
2. ✅ 市场核心
3. ✅ 情绪支持
4. ✅ 形态健康

---

#### 3. 卡位板

**特点**:
- 🎯 板块内超越
- ⚡ 板块切换
- 💰 确定性高

**卡位类型**:
- **日内卡位** - 当日最强
- **板块卡位** - 板块内最强
- **情绪卡位** - 情绪转折点

---

### 打板时机

#### 1. 最佳时间

| 时间 | 机会 | 风险 |
|------|------|------|
| **9:30-10:00** | 强势确认 | 冲高回落 |
| **10:00-11:30** | 换手充分 | 震荡为主 |
| **13:00-14:00** | 午后转强 | 尾盘回落 |
| **14:00-14:30** | 抢筹尾盘 | 情绪博弈 |

---

#### 2. 打板信号

**买入信号**:

| 信号 | 说明 | 操作 |
|------|------|------|
| **涨停封单增加** | 资金抢筹 | 买入 |
| **板块联动** | 板块助攻 | 买入 |
| **情绪转折** | 情绪回暖 | 买入 |
| **突破形态** | 形态确认 | 买入 |

**卖出信号**:

| 信号 | 说明 | 操作 |
|------|------|------|
| **涨停封单减少** | 资金撤离 | 卖出 |
| **板块分化** | 板块走弱 | 卖出 |
| **情绪转弱** | 市场回落 | 卖出 |
| **破板** | 封板失败 | 卖出 |

## 4️⃣ 龙头识别

### 龙头特征

> "看得眼花缭乱也不知道龙头在哪里。"

#### 1. 板块内龙头

**识别方法**:

| 特征 | 说明 |
|------|------|
| **率先涨停** | 板块第一 |
| **封单最大** | 资金最强 |
| **带动性最强** | 板块跟随 |
| **持续性最好** | 连板最多 |

---

#### 2. 市场总龙头

**识别方法**:

| 特征 | 说明 |
|------|------|
| **涨停最早** | 引领市场 |
| **涨幅最高** | 弹性最大 |
| **成交量最大** | 资金最活跃 |
| **跟风最强** | 带动最多 |

---

### 龙头战法

#### 1. 介入时机

**最佳时机**:
1. 🔥 龙头首板
2. 🚀 龙头2板确认
3. 💪 龙头分歧转一致
4. 🎯 龙头反包

---

#### 2. 仓位配置

| 阶段 | 仓位 | 理由 |
|------|------|------|
| **首板试错** | 10-20% | 确定性低 |
| **2板确认** | 30-50% | 确定性中 |
| **3板以上** | 50-80% | 确定性高 |
| **高位博弈** | 递减 | 风险增加 |

---

## 5️⃣ 热点捕捉

### 热点特征

> "热点在哪里？"

#### 1. 热点形成条件

| 条件 | 说明 |
|------|------|
| **题材新颖** | 故事动人 |
| **政策支持** | 利好不断 |
| **资金认可** | 大单流入 |
| **板块联动** | 多股涨停 |

---

#### 2. 热点持续性

| 级别 | 持续时间 | 操作策略 |
|------|---------|---------|
| **超级热点** | >2周 | 重仓参与 |
| **大热点** | 1-2周 | 积极参与 |
| **普通热点** | 3-5天 | 小仓位 |
| **一日游** | 1天 | 不参与 |

---

### 热点操作

#### 1. 介入时机

| 阶段 | 机会 | 风险 |
|------|------|------|
| **启动期** | 追涨/打板 | 追高被套 |
| **发酵期** | 板块内轮动 | 追涨杀跌 |
| **高潮期** | 减仓/止盈 | 追涨被套 |
| **退潮期** | 空仓观望 | 接盘被套 |

---

#### 2. 板块选择

**首选**:
- ✅ 新题材
- ✅ 政策利好
- ✅ 资金流入
- ✅ 形态突破

**规避**:
- ❌ 老题材
- ❌ 利好兑现
- ❌ 资金流出
- ❌ 形态破位

## 6️⃣ 打板实战案例

### 案例1：弱强手法

> "昨日表现弱势,今日转强,形成强烈反差。"

**案例**:
- 皇台酒业
- 大有能源
- 蓝丰生化
- 合富中国

**操作要点**:
1. 🎯 关注昨日弱势股
2. 📊 今日放量转强
3. 🚀 突破均线压力
4. ⚡ 打板介入

---

### 案例2：小强手法

> "围绕主线,看强做次强,最强标的无法介入时,转向次强替代。"

**案例**:
- 脑机方向: 创新医疗买不到 → 转做嘉事堂
- AI应用: 利欧股份买不到 → 转做省广集团
- 智能电网: 汉缆股份买不到 → 转做广电电气

**操作要点**:
1. 🎯 确认主线板块
2. 🚀 锁定最强标的
3. 💡 次强替代方案
4. ⚡ 及时切换

---

### 案例3：中强手法

> "在行情分歧阶段,主动切换至次强。"

**操作要点**:
1. 📊 观察情绪分歧
2. 🚀 识别次强机会
3. ⚡ 及时切换
4. 💰 获取确定性收益
