# 📖 上篇：K线基础知识

## 1️⃣ K线基础回顾

### K线构成

| 组成部分 | 说明 |
|----------|------|
| **实体** | 开盘价与收盘价之间 |
| **上影线** | 最高价与实体上端之间 |
| **下影线** | 最低价与实体下端之间 |
| **颜色** | 上涨红色/绿色，下跌绿色/红色 |

---

### K线的含义

| K线类型 | 含义 |
|----------|------|
| **大阳线** | 多方强势，全面上涨 |
| **大阴线** | 空方强势，全面下跌 |
| **上影线长** | 上方有压力，冲高回落 |
| **下影线长** | 下方有支撑，低位反弹 |
| **十字星** | 多空平衡，方向选择 |

---

## 3️⃣ K线组合形态（上）

### 顶部组合

#### 1. 乌云盖顶

**形态**:
```
   ┌─────┐
   │ 阳  │ 上涨
   └─────┘
      ┌─────┐
      │ 阴  │ 下跌
      └─────┘
```

**特征**:
- 第一天: 大阳线
- 第二天: 高开低走，收盘在第一天阳线实体50%以下
- 第三天: 确认下跌

**操作**:
- 🚨 风险信号
- ✅ 减仓或卖出
- ⚠️ 不要追涨

---

#### 2. 吊颈线（上吊线）

**形态**:
```
      ┌───┐
      │   │ 上影线短
   ┌──┴──┐
   │ 实体 │ 小实体
   └──┬──┘
      │
      └── 下影线长
```

**特征**:
- 实体在顶部
- 下影线是实体的2倍以上
- 上影线很短或没有

**操作**:
- 🚨 顶部信号
- ✅ 减仓
- ❌ 不追高

---

#### 3. 黄昏星

**形态**:
```
   ┌─────┐
   │ 阳  │ 上涨
   └─────┘
      ┌─┐
     ┌┤ ├┐ 十字星
     └┴─┴┘
      ┌─────┐
      │ 阴  │ 下跌
      └─────┘
```

**特征**:
- 第一天: 大阳线
- 第二天: 十字星
- 第三天: 大阴线

**操作**:
- 🚨 强烈见顶信号
- ✅ 果断卖出
- ❌ 不要买入

---

#### 4. 穿头破脚

**形态**:
```
   ┌─────┐
   │ 阳  │ 上涨
   └─────┘
   ┌─────┐
   │ 阴  │ 下跌
   └─────┘
```

**特征**:
- 第二天阴线实体完全覆盖第一天阳线实体
- 成交量放大

**操作**:
- 🚨 强烈反转信号
- ✅ 立即卖出
- ❌ 不要抄底

## 5️⃣ K线常见误区

### 误区1: 单一K线判断

> "坦白说,我过去也踩过不少坑。最常见的误区,就是把布林带上下轨当成绝对的买卖点。"

**错误**:
- ❌ 只看一根K线就做决定
- ❌ 不看成交量配合
- ❌ 不看市场环境

**正确**:
- ✅ K线 + 成交量
- ✅ K线 + 均线
- ✅ K线 + 市场环境

---

### 误区2: 忽视趋势

**错误**:
- ❌ 逆势操作
- ❌ 不看大盘环境

**正确**:
- ✅ 顺势而为
- ✅ 大盘配合

---

### 误区3: 机械套用

**错误**:
- ❌ 机械套用形态
- ❌ 不考虑具体环境

**正确**:
- ✅ 灵活运用
- ✅ 结合实际

## 🔟 K线与均线配合

### 均线配合

| 配合 | 信号 | 操作 |
|------|------|------|
| **K线 + 5日线** | K线站上5日线 | 短期走强 |
| **K线 + 10日线** | 回踩10日线 | 买入机会 |
| **K线 + 20日线** | 突破20日线 | 中期走强 |
| **K线 + 60日线** | 站上60日线 | 长期走强 |

---

### 布林带配合

> "观察价格与中轨的关系。在强势上涨趋势中,价格通常会沿着布林带上半部分,也就是中轨和上轨之间运行。当中轨本身也保持向上态势,而价格回踩到中轨附近并获得支撑时,这可以看做是一个趋势延续过程中的参考点。"

**配合方法**:
| 位置 | 信号 | 操作 |
|------|------|------|
| **回踩中轨** | 支撑有效 | 买入 |
| **跌破中轨** | 支撑失效 | 卖出 |
| **沿上轨运行** | 强势上涨 | 持有 |
| **跌破下轨** | 弱势下跌 | 观望 |
