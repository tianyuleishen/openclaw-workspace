### 系统架构

| 模块 | 功能 |
|------|------|
| 选股因子 | 动量(30%) + 强度(25%) + 量能(20%) + 位置(15%) + 涨跌幅(10%) |
| 买入信号 | 综合评分≥70 + 5日动量>3% + 量比1-3 + 价格位置≥40% |
| 卖出信号 | 综合评分<30 + 5日动量<-5% + 大跌>-5% + 位置<20% |
| 仓位管理 | 最大50%仓位 + 8%止损 + 15%止盈 + 2%单笔风险 |
