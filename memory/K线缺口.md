## 1️⃣ 什么是缺口

### 定义

> "因为股价的K线不是连贯的，会出现突发利空或者利多"

**缺口**是指股价在快速大幅波动中，在K线图上留下的一段没有交易的价格区域。

---

### 缺口的形成

| 因素 | 说明 |
|------|------|
| **突发利好** | 重大利好刺激股价跳空上涨 |
| **突发利空** | 重大利空刺激股价跳空下跌 |
| **资金推动** | 大量资金快速买入或卖出 |
| **市场情绪** | 极端情绪导致的极端波动 |

---

### 缺口的特征

| 特征 | 说明 |
|------|------|
| **跳空** | 开盘价与前一天收盘价不连续 |
| **空白** | K线图上出现空白区域 |
| **突破** | 突破重要阻力或支撑 |
| **信号** | 趋势改变或加速的信号 |

---

### 缺口的意义

| 意义 | 说明 |
|------|------|
| **趋势信号** | 缺口是趋势改变的信号 |
| **力度信号** | 缺口大小反映趋势力度 |
| **持续信号** | 缺口类型预示持续时间 |
| **反转信号** | 竭尽缺口预示趋势结束 |

---

## 2️⃣ 缺口的类型

### 按方向分类

| 类型 | 方向 | 含义 |
|------|------|------|
| **向上缺口** | 跳空上涨 | 多方强势 |
| **向下缺口** | 跳空下跌 | 空方强势 |

---

### 按位置分类

| 类型 | 位置 | 含义 |
|------|------|------|
| **突破缺口** | 趋势起点 | 趋势开始 |
| **持续缺口** | 趋势中段 | 趋势持续 |
| **竭尽缺口** | 趋势末端 | 趋势结束 |

---

### 缺口类型对比

| 对比项 | 突破缺口 | 持续缺口 | 竭尽缺口 |
|--------|----------|----------|----------|
| **出现位置** | 趋势起点 | 趋势中段 | 趋势末端 |
| **出现时机** | 底部或顶部 | 上涨中段 | 上涨末端 |
| **成交量** | 放量 | 持续放量 | 缩量 |
| **回补概率** | 低 | 中 | 高 |
| **操作价值** | 高 | 中 | 低 |

---

## 3️⃣ 突破缺口

### 定义

> "出现向上突破缺口时，应在..."

**突破缺口**是指股价突破重要阻力位或支撑位时出现的缺口。

---

### 突破缺口的特征

| 特征 | 说明 |
|------|------|
| **位置** | 底部或顶部区域 |
| **幅度** | 跳空幅度较大 |
| **成交量** | 成交量明显放大 |
| **不回补** | 短期内不会回补 |

---

### 向上突破缺口

**形态**:
```
   ┌─────┐
   │ 阳  │ 上涨
   └─────┘
      ↑
     缺口
      ↓
   ┌─────┐
   │ 阴  │ 震荡
   └─────┘
```

**含义**:
- 📈 多方强势突破
- 💪 主力资金入场
- 🟢 强烈看涨信号

**操作建议**:
| 建议 | 操作 |
|------|------|
| **买入** | 突破当天买入 |
| **加仓** | 回踩不破买入 |
| **止损** | 跌破缺口止损 |

---

### 向下突破缺口

**形态**:
```
   ┌─────┐
   │ 阴  │ 震荡
   └─────┘
      ↓
     缺口
      ↑
   ┌─────┐
   │ 阴  │ 下跌
   └─────┘
```

**含义**:
- 📉 空方强势突破
- 💀 主力资金出逃
- 🔴 强烈看跌信号

**操作建议**:
| 建议 | 操作 |
|------|------|
| **卖出** | 跌破当天卖出 |
| **减仓** | 反弹不过卖出 |
| **止损** | 反弹突破止损 |

---

### 突破缺口实战技巧

#### 低位突破缺口

**特征**:
- 📍 出现在长期横盘后
- 📊 成交量明显放大
- 🛡️ 缺口短期不回补

**操作**:
- ✅ 突破当天买入
- ✅ 回踩缺口买入
- ✅ 止损设在缺口下方

---

#### 突破缺口买入条件

| 条件 | 要求 |
|------|------|
| **位置** | 相对低位 |
| **成交量** | 放量突破 |
| **幅度** | 跳空3%以上 |
| **收盘** | 收盘价在缺口上方 |

## 4️⃣ 持续缺口

### 定义

> "股价突破并远离密集成交区后出现的第一个缺口为持续缺口"

**持续缺口**是指在趋势发展过程中出现的缺口，表示趋势正在持续。

---

### 持续缺口的特征

| 特征 | 说明 |
|------|------|
| **位置** | 趋势中段 |
| **幅度** | 跳空幅度适中 |
| **成交量** | 持续温和放量 |
| **会回补** | 最终会被回补 |

---

### 持续缺口的类型

#### 向上持续缺口

**形态**:
```
   ┌─────┐
   │ 阳  │ 上涨
   └─────┘
      ↑
     缺口1
      ↑
   ┌─────┐
   │ 阳  │ 继续上涨
   └─────┘
      ↑
     缺口2
      ↑
   ┌─────┐
   │ 阳  │ 继续上涨
   └─────┘
```

**含义**:
- 📈 上涨趋势持续
- 💪 主力持续买入
- 🟢 趋势健康

**操作建议**:
| 建议 | 操作 |
|------|------|
| **持股** | 继续持有 |
| **加仓** | 回调支撑买入 |
| **目标** | 寻找下一个缺口 |

---

#### 向下持续缺口

**形态**:
```
   ┌─────┐
   │ 阴  │ 下跌
   └─────┘
      ↓
     缺口1
      ↓
   ┌─────┐
   │ 阴  │ 继续下跌
   └─────┘
      ↓
     缺口2
      ↓
   ┌─────┐
   │ 阴  │ 继续下跌
   └─────┘
```

**含义**:
- 📉 下跌趋势持续
- 💀 主力持续卖出
- 🔴 趋势较弱

**操作建议**:
| 建议 | 操作 |
|------|------|
| **观望** | 不轻易抄底 |
| **减仓** | 反弹减仓 |
| **止损** | 破位止损 |

---

### 持续缺口的应用

#### 判断趋势持续

| 信号 | 含义 |
|------|------|
| **持续缺口出现** | 趋势还将持续 |
| **缺口间距均匀** | 趋势健康发展 |
| **成交量持续放大** | 主力持续入场 |

---

#### 持续缺口操作

| 操作 | 说明 |
|------|------|
| **持股待涨** | 继续持有 |
| **不追涨** | 等待回调买入 |
| **设目标** | 以下一个缺口为目标 |

## 5️⃣ 竭尽缺口

### 定义

> "股价上升时的最后一个缺口"

**竭尽缺口**是指在趋势末端出现的缺口，表示趋势即将结束。

---

### 竭尽缺口的特征

| 特征 | 说明 |
|------|------|
| **位置** | 趋势末端 |
| **幅度** | 跳空幅度较小 |
| **成交量** | 成交量开始萎缩 |
| **会回补** | 短期内会被回补 |

---

### 向上竭尽缺口

**形态**:
```
   ┌─────┐
   │ 阳  │ 上涨
   └─────┘
      ↑
     缺口
      ↑
   ┌─────┐
   │ 阴  │ 冲高回落
   └─────┘
```

**含义**:
- 📈 上涨最后冲刺
- ⚠️ 多方力量耗尽
- 🔴 即将见顶

**操作建议**:
| 建议 | 操作 |
|------|------|
| **减仓** | 逐步减仓 |
| **止盈** | 锁定利润 |
| **止损** | 跌破缺口止损 |

---

### 向下竭尽缺口

**形态**:
```
   ┌─────┐
   │ 阴  │ 下跌
   └─────┘
      ↓
     缺口
      ↓
   ┌─────┐
   │ 阳  │ 超跌反弹
   └─────┘
```

**含义**:
- 📉 下跌最后冲刺
- 🟢 空方力量耗尽
- ✅ 即将见底

**操作建议**:
| 建议 | 操作 |
|------|------|
| **关注** | 关注止跌信号 |
| **不抄底** | 等待底部确认 |
| **观察** | 观察成交量变化 |

---

### 竭尽缺口的判断

#### 判断方法

| 方法 | 说明 |
|------|------|
| **位置判断** | 连续上涨后出现 |
| **成交量** | 成交量开始萎缩 |
| **K线形态** | 出现顶部K线形态 |
| **技术指标** | RSI超买 |

## 6️⃣ 普通缺口

### 定义

> "普通缺口通常可以分为普通下降缺口和普通上升缺口"

**普通缺口**是指在盘整区域或波动较小的区域出现的缺口。

---

### 普通缺口的特征

| 特征 | 说明 |
|------|------|
| **位置** | 盘整区域 |
| **幅度** | 跳空幅度较小 |
| **成交量** | 成交量无明显变化 |
| **快速回补** | 短期内会被回补 |

---

### 普通缺口的类型

#### 普通向上缺口

**特征**:
- 📈 小幅跳空上涨
- 📊 成交量无明显变化
- 🔄 短期内会回补

**操作**:
- ⚠️ 无操作价值
- 📊 观察即可

---

#### 普通向下缺口

**特征**:
- 📉 小幅跳空下跌
- 📊 成交量无明显变化
- 🔄 短期内会回补

**操作**:
- ⚠️ 无操作价值
- 📊 观察即可

---

### 普通缺口与突破缺口的区别

| 对比项 | 普通缺口 | 突破缺口 |
|--------|----------|----------|
| **位置** | 盘整区域 | 底部或顶部 |
| **幅度** | 较小 | 较大 |
| **成交量** | 无明显变化 | 明显放大 |
| **回补时间** | 快速回补 | 短期不回补 |
| **操作价值** | 无 | 有 |

## 🔟 缺口常见问题

### 问题1：缺口一定会回补吗？

**回答**:
- ✅ 竭尽缺口会回补
- ✅ 持续缺口最终会回补
- ❌ 突破缺口短期内不回补

---

| 区分方法 | 突破缺口 | 持续缺口 |
|----------|----------|----------|

| **成交量** | 明显放大 | 持续放量 |
| **幅度** | 较大 | 适中 |

---

### 问题3：缺口操作要注意什么？

**回答**:
| 注意项 | 说明 |
|--------|------|
| **成交量** | 必须放量 |
| **位置** | 判断缺口类型 |
| **止损** | 必须设止损 |
