# 自我进化技能

## 描述
AI Agent自主进化系统，能够自动学习、诊断、优化、改进自身能力。

## 功能
- 自我诊断 - 分析当前能力状态
- 性能优化 - 自动优化代码和流程
- 能力扩展 - 学习新技能
- 代码重构 - 持续改进代码质量
- 持续进化 - 基于反馈的迭代改进

## 使用方法
```bash
python3 /home/admin/.openclaw/workspace/self_evolution_skill.py
```

## 文件
- self_evolution_skill.py - 主技能
- auto_optimizer.py - 自动优化器
- capability_tracker.py - 能力追踪
