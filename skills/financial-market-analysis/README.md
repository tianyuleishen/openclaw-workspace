# Financial Market Analysis

Precision Financial Insights - Analyze stocks, companies, and market sentiment using authoritative data.

## Features

- Stock analysis with Yahoo Finance data
- Market sentiment analysis
- Company financial insights
- Investment ratings

## Usage

```bash
# Run analysis
python3 main.py

# Or use the agent directly
/market "AAPL"    # Analyze Apple stock
/market "TSLA"    # Analyze Tesla stock
/market "中国平安"  # Analyze Chinese company
```

## Integration

This skill provides financial market analysis capabilities for the AI agent system.
