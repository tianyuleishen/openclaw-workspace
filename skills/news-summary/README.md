# News Summary

Fetch news updates, daily briefings, and global events from trusted international RSS feeds.

## Features

- ✅ International news from RSS feeds
- ✅ Daily briefings
- ✅ Voice summary generation (TTS)
- ✅ Multi-source aggregation

## Usage

```bash
# Get daily news
python3 news_summary.py

# Fetch specific category
python3 news_summary.py --category=technology

# Generate voice summary
python3 news_summary.py --voice
```

## RSS Sources

| Source | URL |
|--------|-----|
| BBC World | https://feeds.bbci.co.uk/news/world/rss.xml |
| BBC Top | https://feeds.bbci.co.uk/news/rss.xml |
| Reuters | https://www.reutersagency.com/feed/ |

## Use Cases

- Daily news briefing
- Industry news monitoring
- Global events tracking
- Voice summaries for commute

Works with OpenClaw's TTS system for voice summaries.
