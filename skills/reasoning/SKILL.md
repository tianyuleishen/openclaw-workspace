# 推理引擎技能

## 描述

自动推理引擎，集成到所有会话中。

## 功能

- 自动检测问题类型
- 调用对应的推理引擎
- 返回验证的答案

## 配置

```json
{
  "reasoning_engine": {
    "version": "5.9",
    "auto_import": true,
    "auto_detect": true
  }
}
```

## 支持的问题类型

| 类型 | 关键词 | 推理引擎 |
|------|--------|---------|
| 三角函数 | tan, cos, sin, θ | v5.9 |
| 极值组合 | 最大, 最小, 范围, 格子 | v5.7 |
| 几何 | 三角形, 圆形, 抛物线 | v5.5 |

## 使用方法

```python
from skills.reasoning import solve

# 自动分析
answer = solve("你的数学问题")
```
